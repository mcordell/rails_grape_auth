This is an example repo that demonstrates setting up rails app that integrates
the following:

- [devise][3]
- [devise_token_auth][1]
- [jToker][5]
- [grape_devise_token_auth][4]
- [grape][2]


I primarily use it to test my gem [grape_devise_token_auth][4].

[2]: https://github.com/intridea/grape
[3]: https://github.com/plataformatec/devise

[5]: https://github.com/lynndylanhurley/j-toker
